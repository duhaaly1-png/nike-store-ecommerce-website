# Nike Store E-Commerce Website

A responsive Nike-inspired e-commerce website developed using HTML, CSS, and JavaScript.

## Features

- Product catalog with multiple shoe models
- Interactive product slider
- Product color selection
- Responsive design
- Login and Signup pages
- Shopping and checkout modal
- Modern UI/UX design

## Technologies Used

- HTML5
- CSS3
- JavaScript
